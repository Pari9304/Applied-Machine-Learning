## 📂 Repository Structure

### 1. Student_Performance_EDA.ipynb

**Project Type:** Exploratory Data Analysis (EDA)

**Description:**
Performed exploratory data analysis on a student performance dataset to understand relationships between study habits, previous scores, sleep hours, extracurricular activities, and overall performance.

**Key Tasks:**

* Data loading and inspection
* Descriptive statistics
* Missing value analysis
* Data visualization using Seaborn
* Feature exploration

**Skills Demonstrated:**

* Data Cleaning
* Data Visualization
* Exploratory Data Analysis

**Key Tasks:**

* Data visualization using regression plots
* Model training using Linear Regression

**Skills Demonstrated:**
